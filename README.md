# AICE-26 Symposium Website

Static website for the AICE-26 Symposium on AI, Consciousness and Ethics.
Part of the AISB Convention 2026, University of Sussex, UK, 1–2 July 2026.

## Files

- `index.html` — main page
- `style.css` — custom styles (uses Bootstrap 5 via CDN)

## Hosting on GitHub Pages

1. Create a new repository on GitHub (e.g. `aice-26`)
2. Upload these two files to the repository root
3. Go to **Settings → Pages → Source → Deploy from a branch → main / (root)**
4. Save. Your site will be live at `https://<username>.github.io/aice-26`

No build step required. Bootstrap 5 is loaded from CDN.
